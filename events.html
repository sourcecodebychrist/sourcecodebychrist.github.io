ESCAPE ROOM

Approx. Duration - 1 hour

Description

What if you are trapped in the 8-bit world of Minecraft and the only way back to reality is through locks and puzzles? Break out into the real world by paying attention to everything you see. This game explores 15+ rooms, and the only objective is to escape. All you require to win is keen observation and puzzle solving ability. A 20-minute time slot will be given to each player and will be ranked according to the amount of levels cleared and the time taken. All the players are also provided with six hints all throughout the game at any point of time.

Scoring Criteria

Amount of levels cleared and the time taken to clear them.

Instructions

  1. Download TLAUNCHER

  2. Create an account in the same website

  3. After installing it, login with the account you've just created.

  4. Install 1.8.8 optifine version in the software

  5. Text us the username

  6. Start the game & go to Multiplayer in title screen

  7. Select Direct Connect

  8. Enter the SERVER ADDRESS when communicated.

  9. Join the Server

Requirements
  1. Laptop with basic requirements
  2. WhatsApp (For Communication)
  3. TLauncher (Software to run the game)
  4. High Bandwidth Internet
  5. Google Meet
Important Guidelines
  1. No spamming in group/game.
  2. Screen should be shared during the event.
  3. You can search on the internet for game riddles.
  4. Don’t join the server until you are told to do so

Description

FIDELITY

Approx. Duration - 1 hour

Description
